# Cryptocurrency Market Segmentation using Unsupervised Learning 💰📊

## 📌 Project Overview
This project applies Unsupervised Learning techniques to analyze and cluster cryptocurrencies based on their behavior.

## 🎯 Objective
To group cryptocurrencies based on:
- Mean Return (profit potential)
- Volatility (risk level)

## ⚙️ Techniques Used
- Feature Engineering
- StandardScaler (Normalization)
- K-Means Clustering
- Elbow Method
- PCA (Dimensionality Reduction)

## 📊 Workflow
1. Data Loading (CSV dataset from Kaggle)
2. Feature Engineering (returns, volatility)
3. Scaling
4. Clustering using K-Means
5. Visualization using PCA

## 📈 Results
Cryptocurrencies were grouped into clusters such as:
- High Risk – High Return
- Medium Risk
- Low Risk

## 🛠 Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
